# Aarhus Techno - Hidden Places

A responsive website implementation of the Figma design for Aarhus Techno community.

## Features

- **Fully Responsive Design**: Mobile-first approach with breakpoints for tablet and desktop

## Structure

- `index.html` - Main HTML structure
- `styles.css` - Responsive CSS with mobile-first design
- `institute.html` `turkis.html` `volume-village.html` - HTML subpages
- `subopage.css` - Subpages styles
- `script.js` - JavaScript for interactivity and animations

## Responsive Breakpoints

- **Mobile**: < 480px
- **Tablet**: 480px - 768px
- **Desktop**: 768px - 1200px
- **Large Desktop**: > 1200px

## Sections
### Homepage
1. **Hero Section**: Full-width hero with logos and navigation
2. **Community Section**: Two-column layout with text and image
3. **Attitude Code Section**: Two-column grid with code guidelines
4. **Music Section**: Grid of playlist cards embeded from Spotify
5. **Events Section**: Calendar display embeded from Google Calendar
6. **Footer**: Contact information and collaboration details

### Homepage
1. **Hero Section**: Full-width hero with logo and tittle
2. **About section**: Descriptive text and address/contact details
3. **Gallery section**: Pictures of the place
4. **Upcoming events section**: Event name and cover with facebook event link
6. **Footer**: Contact information and collaboration details

## Usage

Simply open `index.html` in a web browser. No build process or dependencies required.
