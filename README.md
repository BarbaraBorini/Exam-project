# Aarhus Techno - Hidden Places

A responsive website implementation of the Figma design for Aarhus Techno community.

## Features

- **Fully Responsive Design**: Mobile-first approach with breakpoints for tablet and desktop
- **Smooth Animations**: Intersection Observer for fade-in effects on scroll
- **Interactive Navigation**: Smooth scrolling to sections
- **Modern UI**: Dark theme with vibrant accent colors
- **Accessibility**: Focus states and semantic HTML

## Structure

- `index.html` - Main HTML structure
- `styles.css` - Responsive CSS with mobile-first design
- `script.js` - JavaScript for interactivity and animations

## Responsive Breakpoints

- **Mobile**: < 480px
- **Tablet**: 480px - 768px
- **Desktop**: 768px - 1200px
- **Large Desktop**: > 1200px

## Sections

1. **Hero Section**: Full-width hero with logos and navigation
2. **Community Section**: Two-column layout with text and image
3. **Attitude Code Section**: Two-column grid with code guidelines
4. **Music Section**: Grid of playlist cards
5. **Events Section**: Calendar display
6. **Footer**: Contact information and collaboration details

## Usage

Simply open `index.html` in a web browser. No build process or dependencies required.

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes

- Images are loaded from Figma's CDN (valid for 7 days)
- For production, download and host images locally
- Fonts use system fallbacks (Arial) - consider adding custom fonts for exact design match
